📊 Edge Calculation Explained

What is Edge?

Edge represents your advantage over the sportsbook. It's the difference between the true probability of an event happening (based on our analysis) and the implied probability from the betting odds.

The Formula

Step 1: Calculate Implied Probability

If odds < 0: implied_prob = |odds| / (|odds| + 100)
If odds > 0: implied_prob = 100 / (odds + 100)

Example: For odds of -110, implied probability = 110 / (110 + 100) = 52.4%

Step 2: Calculate Edge Percentage

edge = true_probability - implied_probability
edge_percentage = (edge / implied_probability) × 100

Example: If true prob = 60% and implied = 52.4%, edge = 7.6%, edge% = (7.6 / 52.4) × 100 = 14.5%

How We Calculate True Probability

Our models analyze multiple factors to estimate the true probability of a QB throwing a touchdown:

  • QB Performance: Historical TD rate per game (60% weight)
  • Defense Weakness: Opponent's TDs allowed per game (40% weight)
  • Home Field Advantage: +10% boost for home games
  • League Context: How QB/defense compare to league averages

Edge Tiers & Betting Strategy

PASS < 5% edge - Not worth betting
SMALL EDGE 5-10% edge - Conservative bet (15% Kelly)
GOOD EDGE 10-20% edge - Moderate bet (25% Kelly)
STRONG EDGE > 20% edge - Aggressive bet (25% Kelly)

Real Example

Scenario: Patrick Mahomes to throw TD vs weak defense

Our Analysis: True probability = 65% (0.65)

Sportsbook Odds: -150 (implied prob = 60%)

Edge Calculation:

edge = 0.65 - 0.60 = 0.05

edge_percentage = (0.05 / 0.60) × 100 = 8.3%

Result: SMALL EDGE - Consider betting
