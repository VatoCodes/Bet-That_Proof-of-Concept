What is Edge?

Edge represents your advantage over the sportsbook. It's the difference between the true probability of an event happening (based on our analysis) and the implied probability from the betting odds.

The Formula

Step 1: Calculate Implied Probability

If odds < 0: implied_prob = |odds| / (|odds| + 100)
If odds > 0: implied_prob = 100 / (odds + 100)

Example: For odds of -110, implied probability = 110 / (110 + 100) = 52.4%

Step 2: Calculate Edge Percentage

edge = true_probability - implied_probability
edge_percentage = (edge / implied_probability) √ó 100

Example: If true prob = 60% and implied = 52.4%, edge = 7.6%, edge% = (7.6 / 52.4) √ó 100 = 14.5%

How We Calculate True Probability (Step-by-Step)

Anyone can repeat this calculation! Here's the exact formula we use:

Step 1: Calculate QB TD Rate

qb_td_per_game = qb_total_tds √∑ qb_games_played

Example: If QB has thrown 12 TDs in 6 games:
qb_td_per_game = 12 √∑ 6 = 2.0 TDs/game

Step 2: Get Defense TD Rate

defense_tds_per_game = opponent_tds_allowed_per_game

Example: If defense allows 2.5 TDs/game:
defense_tds_per_game = 2.5

Step 3: Calculate Weighted Base Probability

base_prob = (qb_td_per_game √ó 0.6) + (defense_tds_per_game √ó 0.4)

Why 60/40? QB skill matters more than opponent defense

Example: (2.0 √ó 0.6) + (2.5 √ó 0.4) = 1.2 + 1.0 = 2.2

Step 4: Apply Home Field Advantage

adjusted_prob = base_prob √ó (1 + home_boost)
where home_boost = 0.10 if home, 0 if away

Example (home game): 2.2 √ó 1.10 = 2.42

Step 5: Convert to 0.5+ TD Probability

true_probability = adjusted_prob √ó 0.6

Why √ó 0.6? Converts "TDs per game" to "probability of 0.5+ TDs"

Example: 2.42 √ó 0.6 = 1.452 ‚Üí Clamped to 95% max ‚Üí 95% probability

Note: Final probability is clamped between 5% (minimum) and 95% (maximum) for safety

Edge Tiers & Betting Strategy

PASS < 5% edge - Not worth betting

STRONG EDGE > 20% edge - Aggressive bet (25% Kelly)
